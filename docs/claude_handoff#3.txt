**CURRENT CPU ARCHITECTURE**
- Pipeline stages: IF/ID (combined) → ID/EX register → EX → EX/WB register → WB
- Forwarding: EX/WB → EX for both rs1 and rs2
- Branch flushing: two-bubble penalty, pc = id_ex_pc + id_ex_imm
- MUL stall: mul_stall = mul_start || mul_busy (for PC), mul_busy only (for ID/EX)
- CSR file: mtvec (0x305), mepc (0x341), mcause (0x342), 2-bit internal indexing
- CSR forwarding: ex_wb_csr_wr_en + address match → forward ex_wb_csr_wr_data
- Hardware exception write ports on CSR file (hw_wr_en tied to 0, placeholder for exceptions)

**SPI MASTER — CURRENT CODE**
- Single clock domain design — sclk generated internally, edge detected via sclk_reg
- Mode 0 (CPOL=0, CPHA=0) — sclk idles low, MISO sampled on rising edge, MOSI driven on falling edge
- Parameters: DATA_WIDTH=32, CLKDIV=4
- MSB first transmission
- MOSI pre-loaded with data_in[DATA_WIDTH-1] on start
- Testbench: use DATA_WIDTH=8, CLKDIV=4 for readable waveforms
- Test: single transfer, verify MOSI bits MSB first, drive known MISO pattern, check data_out after done, verify CS and done behaviour
